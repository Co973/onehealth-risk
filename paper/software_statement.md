# Software Statement

One Health Risk is a research-oriented Python package and command-line tool for local One Health surveillance modelling workflows. It is intended for reproducible software evaluation, local validation, and governance-aware experimentation.

The software provides:

- Local-only CLI workflows with no network dependency for core commands.
- Synthetic reviewer data generation.
- Configurable feature groups for clinical, environmental, animal, and geography variables.
- Baseline model training, evaluation, local calibration, and prediction export.
- Reproducibility reports, model cards, and manuscript-ready CSV tables.
- Security-aware path auditing and identifier-column warnings.

The software does not provide clinical validation, diagnostic decision support, compliance certification, or protection against a compromised local machine.
